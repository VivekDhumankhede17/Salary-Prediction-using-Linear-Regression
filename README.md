## **Abstract**

Predicting salaries based on experience and other factors is a common problem in data analytics and human resource management. This paper presents a Salary Prediction system using Machine Learning, specifically applying Linear Regression. The model learns the relationship between input features such as years of experience and outputs an estimated salary. The system helps organizations and individuals make informed decisions regarding compensation. Experimental results show accurate predictions with minimal error.
